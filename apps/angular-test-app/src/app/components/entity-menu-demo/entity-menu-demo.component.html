Description

The entity-menu component is a floating panel containing list of options.

it takes mainly a data (datasource) and a title as input and displays the options in cards.

If you want to :

- Match a datasource column with one of your template data, you should provide columnsMatcher input.

- Change the abbreviation length of the options' title, you should modify the abbreviationLength input.

Inputs

Property Type Default Value Description
title string Title the menu's title
data any[] An array of any data
property string A string representing the correspondance between a data column and the component's template data
abbreviationLength number 1 The number of initials you want to extract from the title
