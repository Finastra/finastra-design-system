## Usage

### Import

```
npm i @finastra/tabs
```

```ts
import '@finastra/tabs';
...
<fds-tab-group>
    <fds-tab-item label="Tab 1">Content tab 1</fds-tab-item>
    <fds-tab-item label="Tab 2">Content tab 2</fds-tab-item>
</fds-tab-group>
```
