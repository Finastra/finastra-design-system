The `<fds-menu-trigger>` is a toggle button used for menu.


## Usage

### Import

```
npm i @finastra/menu-trigger
```

```ts
import '@finastra/menu-trigger';
...
<fds-menu-trigger label="Develop"></fds-menu-trigger>
```
