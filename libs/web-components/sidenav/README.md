Simple wrapper for [Material's Drawer web component](https://material-components.github.io/material-web/demos/drawer/).

## Usage

### Import

```
npm i @finastra/sidenav
```

```ts
import '@finastra/sidenav';
...
<fds-sidenav type="">
   <div slot="sidenavContent">
    Sidenav content
  </div>
  <div slot="navigation">
    Navigation content
  </div>
  <div slot="appContent">
    App Content
  </div>
</fds-sidenav>
```
