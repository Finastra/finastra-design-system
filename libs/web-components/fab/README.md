## Usage

The FAB (aka Floating Action Button) is a button that appears in front of all screen content.

### Import

```
npm i @finastra/fab
```

```ts
import '@finastra/fab';
...
<fds-fab extended gradient label="Edit" icon="edit"></fds-fab>
```
