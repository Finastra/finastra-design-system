# FDS Theme Typography

<Meta title="Foundations/Typography" />

## Include open fonts

We are using Roboto and Spartan Google Fonts in the FDS theme typography.

Make sure to copy the code below into the `<head>` of your html.

```HTML
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;500&family=Spartan:wght@800&display=swap" rel="stylesheet" />
```

## Mixins

```SCSS
@mixin typography($style) {}
```

Where `$style` is any of the [available text styles](#available-text-styles).

### Example

```SCSS
@use '@finastra/fds-theme' as fds;

.my-component-subtitle {
  @include fds.typography(subtitle-2);
}
```

## Helper classes

Before being able to use typography helper classes, you will have to include them in your Sass styles.

```SCSS
@use '@finastra/fds-theme/helpers';
```

### Example

```HTML
<h1 class="fds-headline-1">This is headline 1</h1>
```
